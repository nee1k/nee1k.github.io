API for Processing Connected Car Data

Project Overview

In a multifaceted role, I played a pivotal part in enhancing data-driven insights within the automotive domain. Leveraging my expertise, I devised GraphQL APIs tailored to extract automobile sensor data over the last 90 days. This facilitated comprehensive data analysis and enabled advanced functionalities such as pothole detection. Furthermore, I contributed significantly to optimizing infrastructure costs by implementing strategic measures. By configuring AWS S3 policies based on usage through Terraform and implementing CI/CD pipelines, I reduced storage expenses by an impressive 35%. Additionally, I streamlined data processing workflows by scripting Spark jobs to handle and analyze generated parquet files efficiently. These efforts culminated in the seamless construction of API payloads for 13 distinct data requests, contributing to a more efficient and cost-effective data processing ecosystem.
